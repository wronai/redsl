# 07 — PyQual: analiza jakości kodu

Demonstracja modułu `redsl pyqual` — statyczna analiza jakości kodu Python
z użyciem AST (wykrywanie nieużywanych importów, magic numbers, brakujących docstringów).

# CLI
redsl example pyqual
redsl example pyqual --scenario advanced

# Skrypt
python main.py
python main.py --scenario advanced
```
